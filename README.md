# Quick GPT Selected (Text)

Simple Firefox plugin that does Quick GPT query of selected text in the browser.

## Installation

Download the [xpi](https://github.com/ikanher/quick-gpt-selected/blob/master/quick_gpt_selected-0.5-fx.xpi?raw=true).

Go to 'about:addons' in Firefox. Click the settings wheel. Choose 'Install Add-on From File'. Navigate to xpi and click 'Open'.

Go to properties of the addon and add your OpenAI key, and configure possible prompts.

## Usage

Select text in a page, right-click, and choose a prompt from the Quick GPT Selected menu.
The response streams inline near your selection. Click "More" in the inline card to open a
popup with a detailed response.
